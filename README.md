## 🎨 设计规范

本项目采用统一的Shopify风格设计系统。在创建新模块或页面前，请务必阅读：

**[DESIGN_GUIDELINES.md](./DESIGN_GUIDELINES.md)** - 完整的设计规范文档

**核心设计原则：**
- **双层导航结构**：一级导航（下划线式高亮）+ 二级导航（胶囊式按钮）
- **现代化Shopify风格**：扁平化设计、充足留白、卡片式布局
- **统一视觉语言**：黑色系主色调、清晰的层级关系

**参考实现：**
- 产品编辑页面（`/client/src/pages/ProductEdit.tsx`）
- 导航组件（`/client/src/components/ProductEditTabs.tsx`）

所有新模块必须遵循设计规范，确保系统的一致性和专业性。
