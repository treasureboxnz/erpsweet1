## 📊 数据库实体关系（ERD）

```
Product（母产品）
  ↓ 一对多
Variant（批次/变形）
  ├─ 一对多 → Variant-Pricing（批次定价）
  ├─ 一对多 → Variant-Pricing-History（定价历史）
  ├─ 一对多 → Variant-Images（批次图片）
  ├─ 多对多 → Customers（通过variant_customer_links）
  └─ 一对多 → Variant-Suppliers（批次-供应商版本）
       ↓ 一对多
     OrderLine（订单行）
```
